Общ вид на дефиницията на даден клас:
class Name ty where
... signature involving the type variable ty ...

Декларацията включва името на класа и т. нар. signature (сигнатура; „подпис“) на класа, т.е. списък от имената и типовете на функциите, които еднозначно го определят - това са функциите, които следва задължително да бъдат дефинирани за всички типове, които са негови екземпляри.

Пример 1.
class Visible a where
    toString :: a -> String
    size :: a -> Int

Пример 2.
class Eq a where
    (==), (/=) :: a -> a -> Bool
    x /= y = not (x==y)
    x == y = not (x/=y)
